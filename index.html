jquery-afterresize

Catch a JavaScript event when resizing window ends

With jQuery afterresize event plugin, an event is triggered when resizing the window has ended.

You can also bind the afterresize event to a DOM element, but it requires Ben "Cowboy" Alman's jQuery resize plugin.

Basic usage:

$(window).bind("afterresize", function() {
    // your code goes here
});

or:

$(window).afterresize(function() {
    // your code goes here
});
